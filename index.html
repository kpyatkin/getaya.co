For Windows 7 or later. Installer supports both 32 and 64-bit operationg systems.

Aya is a well-designed tool that analyzes disk usage. It displays the largest files and folders in selected locations which allows you to see what files are taking up the most disk space. The app has fully featured built-in duplicate finder that helps you locate and remove identical files on your disk.

Big files are prioritized

By default, Aya shows you the largest files and folders sorted by size in descending order. This allows you to determine what files are taking up the most disk space. You can choose Top 10, 50, 100 etc. largest files.

Built-in duplicate finder

The app contains fully featured duplicate finder, that uses advanced algorithms to compare not only file names, but also contents to ensure no false search results. There is no need to worry, Aya will take care that at least one instance of each file will be kept safe.

Explorer-like

You can move files or folders to the recycle bin by pressing Del or delete them completely by Shift+Del just as it is done in Explorer. You can also perform a lot of other file actions using context menus and keyboard shortcuts.

Auto Update

Software auto update feature allows you not to worry about manual update checking and downloading setup file. When update will be ready, all you have to do is just click button.

Full Folders Duplicates

Duplicate files are in folders like usual files. Aya detects such folders and show them if they are whole duplicated

Content Filters

It is much simpler to manage files as categories. Thanks to content filters, you can easy choose only music files, or music and video and so on. You can also create your own filter based on file types you want.

Feedback Reporting

Your feedback is valuable to us. If you have problems with your license, want to request a feature, or report a bug, we're more than happy to help. You can do that without leaving an application.

Cross Drive Scanning

Aya has flexible scan source selecting. You can choose any drive or and any folder at any combination for scan starting
